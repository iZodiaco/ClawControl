// capabilities

Everything you need to
command your AI

Built for power users who need full control over their OpenClaw agents, sessions, and workflows.

Concurrent Streaming

Talk to multiple agents simultaneously. Per-session stream isolation means zero cross-contamination between conversations.

Server Settings

Full-page editor for your OpenClaw server configuration. Agent defaults, tools, memory backends, and channel policies.

Subagent Spawning

Spawn isolated subagent sessions for parallel task execution. Inline status blocks with popout windows for each subagent.

ClawHub Skills

Browse, search, and install skills from ClawHub. VirusTotal security scan badges, download counts, and one-click install.

Thinking Mode

Toggle extended thinking for complex tasks. See the agent's reasoning process displayed inline as it works through problems.

Tool Visibility

See every tool call inline as the agent works. Full transparency into code execution, web searches, and file operations.

Agent Management

Create, configure, and delete agents. Browse agent profiles, workspace files, and per-session identity switching.

Cron Jobs

View and manage scheduled tasks with live status updates. Pause, resume, and inspect cron job details and run history.

Cross-Platform

Runs everywhere. Windows and macOS via Electron, iOS and Android via Capacitor. One codebase, full native feel.
